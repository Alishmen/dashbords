# Декомпозиция данных - Дашборды

Приложение для визуализации данных о заболеваемости и смертности с интерактивной фильтрацией.

## Технологии

- **React 18.2.0** с **TypeScript**
- **Material-UI (MUI) 5.15.15** - UI компоненты
- **Recharts 2.10.3** - библиотека для визуализации графиков
- **React Router DOM 6.22.3** - навигация между дашбордами
- **Create React App** - инструменты сборки

## Установка и запуск

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm start
```

Приложение откроется на [http://localhost:3001](http://localhost:3001)

### Сборка для production

```bash
npm run build
```

## Дашборды

### 1. Заболеваемость. ДН

- Пирамида заболеваемости по полу и возрасту
- Пирамида охвата ДН (диспансерного наблюдения)
- Датчики показателей:
  - Охват ДН
  - Полнота охвата ДН
  - Преемственность
  - Своевременность

### 2. Структура смертности

- Число умерших по полу и возрасту
- Половозрастная структура по месту смерти
- Структура обращаемости среди умерших вне стационара
- Общебольничная и досуточная летальность

### 3. Общий показатель смертности

- Таблица смертности по медицинским организациям
- Половозрастная структура населения
- Рейтинг групп заболеваний по числу умерших (с кривой Парето)
- Рейтинг медицинских организаций по смертности (с кривой Парето)

## Фильтрация

Все дашборды поддерживают интерактивную фильтрацию по:
- Территории
- Медицинской организации (МО)
- Участку
- Полу
- Возрасту
- Нозологической группе

## Особенности

- Полностью типизированный код на TypeScript
- Адаптивный дизайн
- Цветовая индикация статусов (красный/желтый/зеленый)
- Интерактивные графики с подсказками
- Современный UI на Material-UI
