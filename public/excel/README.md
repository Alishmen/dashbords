# Папка для Excel файлов

Эта папка предназначена для хранения Excel файлов (.xlsx, .xls), которые будут использоваться в дашбордах.

## Структура файлов

Рекомендуемые названия файлов:
- `заболеваемость.xlsx` - файл с данными заболеваемости
- `охват_дн.xlsx` - файл с данными охвата диспансерного наблюдения
- `датчики.xlsx` - файл с показателями ДН (опционально)

## Формат данных

Excel файлы должны содержать листы:
- **Мужчины** - данные для мужчин
- **Женщины** - данные для женщин

Структура данных в каждом листе:
- Колонка A: Возрастная группа (например, "0-1", "1-4", "5-9" и т.д.)
- Колонка B: Первичная заболеваемость/охват
- Колонка C: Общая заболеваемость/охват

## Доступ к файлам

Файлы из этой папки доступны по пути: `/excel/название_файла.xlsx`
